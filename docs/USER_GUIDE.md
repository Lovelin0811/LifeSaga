# LifeSaga 使用说明

## 核心功能

| 功能 | 说明 |
|---|---|
| 创建副本 | 选择类型并填写名称，可选封面和简介 |
| 编辑副本 | 修改名称、类型、封面、简介、公开状态 |
| 节点记录 | 为副本添加标题、描述、地点、时间、照片、里程碑 |
| 成就系统 | 根据行为自动解锁成就并累计经验值 |
| 我的相册 | 展示副本封面图与节点照片 |
| 数据统计 | 展示副本数、节点数、完成情况等统计 |

## 副本类型

| 枚举值 | 中文含义 |
|---|---|
| `LIFE` | 生活 |
| `TRAVEL` | 旅行 |
| `STUDY` | 学习 |
| `WORK` | 工作 |
| `HEALTH` | 健身 |
| `CREATIVE` | 创作 |

## 稀有度规则

| 节点数 | 稀有度 |
|---|---|
| `0-2` | `COMMON` |
| `3-5` | `UNCOMMON` |
| `6-10` | `RARE` |
| `11-20` | `EPIC` |
| `21-29` | `LEGENDARY` |
| `30+` | `MYTHIC` |

## 公开副本说明

| 场景 | 当前行为 |
|---|---|
| 自己查看自己的公开副本 | 返回完整详情 |
| 查看别人的公开副本 | 仅返回裁剪后的节点内容，图片可预览 |
| 未公开副本 | 只有所有者可查看 |

## 登录说明

| 项 | 说明 |
|---|---|
| 登录方式 | 小程序启动后自动登录 |
| 本地开发 | 可启用 `auth.dev-login-enabled=true` |
| 线上环境 | 必须配置真实微信 `app-id` 与 `app-secret` |
