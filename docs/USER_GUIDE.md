# LifeSaga 使用指南

## 这是什么

LifeSaga（人生副本）是一个记录人生经历的小程序。它把恋爱、旅行、成长等经历变成 RPG 游戏里的"副本"，每段回忆都是一个"副本"，副本里的大事小事就是"节点"。

## 怎么用

### 首页

打开小程序就是首页，展示你创建过的所有副本。每个副本卡片上会显示：

- 副本名称和封面图
- 稀有度标签（普通 / 稀有 / 史诗 / 传说 等）
- 节点数量

底部有三个统计数字：进行中的副本数、已完成的副本数、所有副本的节点总数。

点击任意副本卡片进入详情页。

### 创建副本

点底部的「+」按钮进入创建页：

1. **选择类型** — 6 种副本类型可选：恋爱、旅行、成长、学习、工作、自定义
2. **填写信息** — 副本名称、描述
3. **上传封面** — 给副本配张封面图
4. **保存** — 一个全新的副本就创建好了

### 副本详情

进入副本后，按时间顺序展示所有节点。可以：

- 往下翻看历史记录
- 点右上角「编辑」修改副本信息
- 点右上角「添加节点」往副本里加新内容

### 添加节点

在副本里点「添加节点」：

- **标题** — 简短描述这件事
- **内容** — 详细记录（可选）
- **地点** — 发生在哪里
- **时间** — 什么时候发生的
- **照片** — 上传配图
- **里程碑** — 如果是重要事件，勾上，时间线上会突出显示

### 个人中心

底部「我的」页面展示：

- 头像和昵称（自动同步微信）
- 已解锁的成就徽章
- 总副本数、总节点数

## 稀有度是怎么回事

每个副本根据**节点数量**自动评定稀有度，越丰富的副本等级越高：

| 节点数 | 稀有度 | 颜色 |
|--------|--------|------|
| 0 - 2 | 普通 | 灰色 |
| 3 - 5 | 稀有 | 绿色 |
| 6 - 9 | 史诗 | 紫色 |
| 10 - 14 | 传说 | 金色 |
| 15+ | 神话 | 渐变色 |

多往副本里加节点，稀有度会自动升级。

## 成就系统

完成特定操作会自动解锁成就徽章，比如：

- 创建第一个副本
- 添加第一个节点
- 节点数达标
- 标记里程碑
- 等等...

去「我的」页面查看已解锁的成就。

## 常见问题

**需要注册吗？**
不需要。微信扫码打开即自动登录，不用填账号密码。

**数据安全吗？**
数据存在你自己的账号下，只有你能看到。服务器用 HTTPS 加密传输。

**怎么删除副本或节点？**
在详情页长按节点可以删除，副本编辑页有删除按钮。

**为什么节点数增加了但稀有度没变？**
稀有度是分段的，比如 3 到 5 个节点都是"稀有"，再多加一个就会升级到"史诗"。
